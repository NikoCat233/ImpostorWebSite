If you would like Custom servers to play TOWN OF HOSTS MODS On, Follow these instructions

Windows

Download This File Here, Then Just double click on it and the custom servers will be added to your game and you will never have to repeat this step again!

If the above method does not work for you, Download the regioninfo.json Here, You may need to right click on the page and choose "Save as other" to mannually download the json file. Then press Win + R and enter & run the following command:

%userprofile%\AppData\LocalLow\Innersloth\Among Us

Replace regioninfo.json with the one you just downloaded and restart your game.

If you have RegionInstall, You can download the config file Here, Then replace the config in Among Us\BepInEx\config with the new one

* - If you have other mods Like TOUR, TOR or Las Monjas, You Already have these servers and dont need to do anything as the servers are already in your game
