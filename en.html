Install Instructions - 安装指南

This page helps you to Install NikoCat233 's Custom Region to your Among Us game.

中国玩家请访问这个链接查看安装教程: https://au.niko233.me/cn.html

Windows Desktop Users

Town of Host Enhanced has already built in Niko233 's Custom Region

Method 1 : Download the install script and run it

Very Simple! Just download the bat file below and click it, then reboot your game. You should be able to see the custom regions

https://au.niko233.me/Setup_Custom_Server.bat

Not working? Try Method 2

Method 2 : Manually install regioninfo.json

First, download regioninfo.json from the following link

https://au.niko233.me/regioninfo.json

Some browser will try to open the above file as new page. Just right click the page and choose "Save As", save it to regioninfo.json

Then, press Win + R or open "Run" in your start menu. Input the following link (click on it to copy to clipboard!)

%userprofile%\AppData\LocalLow\Innersloth\Among Us

Input it and click "Run". It will open your AmongUs Data folder, copy the regioninfo.json you just downloaded to replace the existing old one.

Finally, reboot your game and you should see the custom regions.

Older version of Among Us

If you are playing a Older Version of Among Us that supports DnsRegionInfo, you can just click and copy the following link to your game ip input window.

Direct version for older version of Among Us (2021.11.9 - 2022.10.25)

NA_US : 354na.niko233.me:22031 or 354us.niko233.me:22031

AS_CN : 354as.niko233.me:22031 or 354cn.niko233.me:22031

Extra method: Mini.RegionInstall Mod

You can download config file here if you have Mini.RegionInstall plugin installed.

https://au.niko233.me/at.duikbo.regioninstall.cfg

Copy it to your BL config folder (Among Us\BepInEx\config) and replace the old config there.

Android | Iphone players

If you can normally login to your among us account, just do step one.

If you are using Gspace or OurPlay, please follow step 2 first.

Step 1 : Click the buttons

Open your Among Us and keep it running background. Then open your browser and click the following buttons.

On every click it should switch to your Among Us and install a single region. Keep clicking and you can install all regions.

Keep in mind that you have to keep among us running while you click these buttons

Running Among Us in vitural environment like OurPlay or Gspace? Check step 2

Step 2 : for Vitural Environment like OurPlay and Gspace

Your browser can't forward requests to Among Us in vitural environment directly

You need to import your browser into your vitural environment.

Via Browser is easy to use and small in size. Import it into your Gspace / OurPlay

First launch AmongUs in Gspace / OurPlay, then run your browser and open this website page.

Click the buttons and then it should jump to Among Us game and install the region.
