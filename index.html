Windows Desktop

Close Among Us first, then download and run the script below. Open Among Us again after the script finishes.

This script is intended to work with the PC versions from Steam, Epic Games Store, itch.io, and the Xbox app / Microsoft Store. For manual installation, check Method 2 or Method 3 based on your game version.

If the script does not take effect, right-click it and choose "Run as administrator".

Setup_Custom_Server.bat

  1. Download regioninfo.json.
  2. Press Win + R and open %userprofile%\AppData\LocalLow\Innersloth\Among Us.
  3. Replace the existing `regioninfo.json` file, then restart the game.

If Method 1 does not work for the Microsoft Store version, follow these manual steps:

  1. Download regioninfo.json.
  2. Open %userprofile%\AppData\Local\Packages\Innersloth.AmongUs_fw5x688tam7rm\SystemAppData\wgs.
  3. Find the folder with long numeric names and open the JSON-like save file inside.
  4. Paste the contents of `regioninfo.json`, save, and restart the game.

If you use Mini.RegionInstall, download at.duikbo.regioninstall.cfg and copy it to Among Us\BepInEx\config.

Windows 桌面端

请先关闭 Among Us，再下载并运行下面的脚本。脚本安装完成后，再重新打开 Among Us。

这个脚本理论上支持 Steam、Epic、itch.io 和 Xbox 应用 / 微软商店的 PC 版 Among Us。如需手动安装，请根据游戏版本查看方法 2 或方法 3。

如果脚本不生效，请右键脚本并选择“以管理员身份运行”。

Setup_Custom_Server.bat

  1. 下载 regioninfo.json。
  2. 按 Win + R，打开 %userprofile%\AppData\LocalLow\Innersloth\Among Us。
  3. 把下载好的 `regioninfo.json` 替换进去，然后重启游戏。

如果方法 1 在微软商店版上安装失败，再按下面步骤手动安装：

  1. 下载 regioninfo.json。
  2. 打开 %userprofile%\AppData\Local\Packages\Innersloth.AmongUs_fw5x688tam7rm\SystemAppData\wgs。
  3. 进入名称很长的数字文件夹，找到里面类似 JSON 的存档文件。
  4. 把 `regioninfo.json` 的内容粘贴进去，保存后重启游戏。

如果你使用 Mini.RegionInstall，下载 at.duikbo.regioninstall.cfg，并复制到 Among Us\BepInEx\config。
