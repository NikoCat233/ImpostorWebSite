Announcements | 公告

Notice

We disabled the server side force auth. This means that we no longer kick player for "Server cannot Auth you" And these players will not be assigned a friendcode. We still will try to assign friendcodes to authorized players. This update means to allow more players to join the region. However hackers will also be able to join the server with fake accounts. If your room got hacked or you want everyone to have a friendcode, just enable "Kick friend code non exist players" in TOHE options, and those left will all be real players authorized by server.

我们关闭了服务器的强制验证，现在使用加速器和神必Ourplay也能加入服务器，但是无法分配好友id。如果你被挂飘了，只需要打开TOHE设置里面的"踢出没有好友代码的玩家"，剩下的就都是真人玩家了。
